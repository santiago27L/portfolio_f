Dashboard - Modelo SQL

In this project we developed the entity-relationship diagram to design a database for a courier company, therefore, we used a relational database SQL type, manipulated from phpmyadmin. Additionally, the data dashboard was made in Python using the tkinter library. A heat map was also created using a json file and the geopandas library.

ecommerce model, SQL, html, python

In this project we made a model for an automated purchasing system, this was done through an RFID system connected to a local area network through a microcontroller configured as a socket, additionally in this project we made a dashboard in Python through its tkinter library connected to the local server where the relational SQL database is located, also from a web server we made the eccomerce model along with a firebase database, which was synchronized with the modifying operations made from the GUI or from the web.

PowerBI Dashboard - Mongodb

In the development of this project a dashboard was made in the powerBI software. In this project we started from a business model of a courier company, to make a non-SQL database, that is, mongo DB, which had the necessary structure to provide information on the financial status of the company.

3d detergent plant model

In this project, the mechanical design of a detergent powder production, packaging and palletizing plant was made in Solidworks, initially in Solidowrks a motion study was performed, and then the model was imported into a virtual environment in unity where it was connected to a Siemens PLC S7-1500 to automate the designed model.

Aquatic waste collection platform

In this project, an aquatic platform was designed for garbage collection, the system designed was a fiberglass catamaran, driven by remote control, in the communications system was connected to a web page from where you could also control the direction through the mobile network.

Electro & Echocardiograph 3d models

This project proposes the 3D modeling of medical equipment for electrocardiography and echocardiography, with the objective that they can be implemented in a virtual reality environment where medical students can practice the execution of these procedures.

Computer vision and image processing for object classification using artifical neural networks

A user interface was created to load a photograph of the workspace, then using image processing methods to binarize the photo and select the regions of interest, and finally pass them through multilayer perceptron neural networks and hopfield networks to classify the objects.

Interface for the study of manipulator trajectories through kinematics

An interface was developed for the control and study of the kinematic movements of two anthropomorphic manipulators built in PLA, through this interface it is possible to define start and end points monitoring the movements of each degree of freedom.

Solution of the traveling salesman problem, through optimization algorithms and Kohonnen neural networks.

From the detection and classification of three different classes in a workspace, we used hopefield networks and genetic optimization algorithms, to propose the most optimal route of a mobile robot from the starting point to the destination point passing through all the cities of the corresponding class, finally by fuzzy logic to define the internal speed control of the wheels of the mobile robot.
